# Title

Cache Simulator

## Description

C++ code to implement Cache simulation

### Algorithms Implemented
	1) FIFO
	2) LRU
	3) Random

### Requirements

1) C++ code editor and compiler i.e DevC++

2) Trace.csr (File is provided in the project folder)

## Running the tests

	1)Cache size in KBs
	2)Associativity
	3)Block size 

### Run

Just click compile and run
